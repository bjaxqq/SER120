# GradingCalculator

### Instructions
Implement a program that reads points (out of 100) in 5 courses, calculates the average, and prints the grade based on the scale as shown below:

| Grade Range (%) | Grade |
| :--: | :--: |
| 93.00 - 100 | A |
| 90.00 - 92.99 | A- |
| 87.00 - 89.99 | B+ |
| 83.00 - 86.99 | B |
| 80.00 - 82.99 | B- |
| 77.00 - 79.99 | C+ |
| 73.00 - 76.99 | C |
| 70.00 - 72.99 | C- |
| 60.00 - 69.99 | D |
| < 60 | F |
